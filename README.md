# Todo List App

Simple To-do List built with Vue.js 3 with Typescript
All the dates are UTC

Node.js +14
NPM/YARN

## Installation

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint
```
